
Context User Field module for Drupal 7.x.
This module creates a condition for Context module.
Context User Field provides a context condition related to the user field values. 
For example you can show block only to users who set field gender = 'female'.
It allows to add multiply values, each on new line.
For example, you can add a context condition of 'user fields' to show particular 
block only to users who has field 'relationship' with the values 'father' or 'mother'.
It allows to add condition to act on empty or non empty value ('Empty field' checkbox).

REQUIREMENTS
------------
* Install Context module https://www.drupal.org/project/context
* Install Chaos tool suit (ctools) https://www.drupal.org/project/ctools

INSTALLATION INSTRUCTIONS
-------------------------
1. Download module.
2. Copy this module directory to your sites/all/modules.
3. Enable the module.
